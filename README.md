# Описанеи программы
Вводится 2 числа - а и b.
Выводится:
- 1 Сумма этих чисел.
- 2 Разность этих чисел.
- 3 Произведение этих чисел.
- 4 Среднее арифметическое этих чисел.
- 5 Разность максимального и минимального по модулю.
- 6 Частное двух чисел.
